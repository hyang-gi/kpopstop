## How to pull

### How to use Git
0. Run checkout -b "Your branch name"

1. Modify your files
2. Run 'git status' to check the modified files
3. Stage the change: 'git add' + the name of the changed file
4. Run git commit -m "Notes for this change"
5. Run git push
